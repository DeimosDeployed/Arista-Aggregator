
Each folder file includes 10% overlap with the previous folder. Example: 100-tier file includes the last 10 configs from the 50-tier file plus 90 new configs.

## Config Validation

Before saving each config, the system checks:

- vmess protocol: presence of v, ps, add, port, id, aid fields + valid UUID + port range 1-65535
- vless/trojan protocol: presence of @ and # characters in string
- ss protocol: valid base64 encoding + presence of : after decoding + valid port
- Other protocols: matches protocol pattern and no malicious characters

Invalid configs are removed. Their count is reported in the final log.
